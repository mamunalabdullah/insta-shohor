How does JavaScript work?

JavaScript is a high level client side scripting language. It reads codes from top to bottom. It can do one task at a time. To make web page interactive we use javascript. It is single threaded programming language.

What does javascript event loop do?

Event loop is responsible for checking the call stack in the event queue. It is also called as callback queue.

Local Storage VS Session Storage

Local storage is not session based where session storage is session based. Data in Sesson storage is deleted when session ends but local storage data is not deleted when session ends.
